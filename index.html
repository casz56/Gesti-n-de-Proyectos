        Manual de uso

        (1) Modelo INFIHUILA (procedimiento / enfoque estratégico) y (2) Portafolio de Servicios (estructura institucional basada en las páginas 2 y 3).

        1) Navegación general

        • Menú superior: usa los botones para cambiar entre layouts sin recargar la página.
        • Scroll: el contenido es responsive; en pantalla pequeña los bloques se apilan para lectura cómoda.
        • Tarjetas y elementos: pasa el cursor (o toca en móvil) para ver efectos y resaltes.

        2) “Modelo INFIHUILA”

        • Propósito: presentar el enfoque estratégico y el procedimiento paso a paso.
        • Lectura guiada: sigue el orden de los bloques (títulos → explicación → componentes).
        • Identidad visual: los colores y estilos se aplican de forma corporativa para consistencia.

        3) “Portafolio de Servicios”

        • Buscar: escribe palabras clave (ej. “libranza”, “CDAT”, “tesorería”) para ubicar servicios rápido.
        • Filtrar: selecciona una línea (Depósitos, Créditos o Administración de recursos) para depurar resultados.
        • Etiquetas: usa los chips temáticos para explorar por enfoque (ambiental, ecoturismo, espacio público, etc.).
        • Detalle: abre/consulta tarjetas para ver descripción y alcance del servicio.

        Nota: el botón flotante “?” abre este manual en cualquier sección para apoyar la navegación del proyecto completo.
